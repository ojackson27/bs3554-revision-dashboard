# BS3554 Dashboard V2 Redesign Spec

## Goal
Redesign the revision dashboard with a space-purple theme, tab-based navigation, SVG icons, and a task-driven confidence scoring system replacing manual sliders.

## Architecture
Client-side Next.js 14 app, all state in localStorage. Four tabs rendered client-side via state (no routing needed). Confidence scores are derived from task completion — no manual input.

## Visual System
- **Background**: `#100820`
- **Cards**: `#1c0f3a`
- **Borders**: `#5b21b6` / `#3b1f7a`
- **Accent gradient**: `linear-gradient(135deg, #7c3aed, #db2777)` — used on active tabs, primary buttons, progress bars
- **Label colour**: `#c084fc` (soft purple)
- **Text**: `#f8fafc`
- **Muted text**: `#94a3b8`
- **Danger**: `#ef4444` (weak topics)
- **Success**: `#22c55e` (completed tasks)
- **Icons**: Lucide React — zero emojis used as UI icons
- **Branding**: "Cardiff University" in `#c084fc` small caps above module title in header

## Navigation
Sticky top navbar with 4 tabs: Home | Confidence | Notes | Strategy.
Active tab has the purple→pink gradient underline/background. Tab state managed in React (`useState`), no URL routing.

## Data Model

### Tasks (per topic)
```ts
interface TopicTasks {
  lectureSlides: boolean        // 20 pts
  tutorialQuestions: number     // 0–4, 10 pts each (max 40)
  timedQuestion: boolean        // 25 pts
  discussionPlan: boolean       // 15 pts
}
```

### Confidence Score Formula
`score(topic) = (slidesPts + tutorialPts + timedPts + planPts) / 100 * 100`
`overallConfidence = average(all 8 topic scores)`

### localStorage shape
```ts
interface RevisionStore {
  completedSessions: number[]
  tasks: Record<string, TopicTasks>   // keyed by topic ID
  notes: Record<string, string>       // keyed by topic ID
}
```
Manual `confidenceScores` field removed entirely.

## Tabs

### Home Tab
- **Header**: Cardiff University label, module title, live countdown, overall progress bar (sessions), prominent Reset button
- **Current Focus panel**: Next incomplete session, session tip, gradient "Mark Complete" button
- **4-Day Plan**: 2×2 grid of day cards. Each session row has checkbox, time-of-day icon (SVG), title, topics

### Confidence Tab
- **Top band**: Overall confidence % (large), derived from all task scores
- **Mini radar chart** (Recharts) alongside the overall score
- **Topic grid**: 2-col grid of topic cards. Each card shows:
  - Topic name
  - Confidence % (colour-coded: green ≥70, amber 40–69, red <40)
  - Progress bar filled to score %
  - 4 task rows with checkboxes:
    - Read lecture slides (toggle)
    - Tutorial questions (0–4 stepper)
    - Timed exam question (toggle)
    - Discussion plan (toggle)
  - Weak topics (score <40%) get a red border

### Notes Tab
- Accordion list of 8 topics
- Textarea per topic, persisted in localStorage
- Dot indicator if note has content

### Strategy Tab
- Exam format reminder
- Top 3 recommended questions ranked by confidence score (highest first)
- Key reminders list
